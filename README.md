# Repoman

A Git tool for tracking file changes over time.

## Installation

    $ gem install repoman

## Usage

    1. Go to the repo's base directory of a file you are trying to track
    2. Run `repoman file_name_you_are_trying_to_track`
    3. Example `repoman lib/map.rb`
